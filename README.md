# 💝 Сердцевина — Найди свою половину

Сайт знакомств нового поколения с ИИ.

## 🛠 Стек
- Frontend: React + Vite + TailwindCSS
- Backend: Node.js + Express + MongoDB
- AI: GigaChat + DeepSeek
- Деплой: Yandex Cloud + Nginx + PM2

## 🚀 Деплой

```bash
# Клонирование
git clone https://github.com/Budnikcam/serdcevina.git
cd serdcevina

# Бэкенд
cd server && npm install

# AI
cd ../ai-services && pip install -r requirements.txt

# Фронтенд
cd ../client && npm install && npm run build
